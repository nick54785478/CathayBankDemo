架構

MVC 架構

技術棧

  • 後端: Spring Boot、Spring Data JPA
  • 前端: 使用 CDN 掛載 Vue.js 搭配 Vuetify
  • 資料庫: H2

功能介紹

  • 呼叫 Test 進行測試
    • BitcoinFeignClientTest: 透過 Feign Client 取回 Bitcoin 資料
    • BitCoinServiceImplTest: 對 Service 進行測試 (依需求打開 @Test 註解)
    • CathayBankDemoApplicationTests: 透過 Feign Client 對後端進行集成測試 (需先啟動後端專案)
  • 前端頁面 (供參): Vue3 (可能有 Bug，後續若有時間再修改)
    • 新增: 點擊後顯示 Dialog，輸入並送出
    • 呼叫 Coindesk API: 點擊後將資料渲染到頁面
    • 依幣別查詢、修改、刪除:
      • 查詢: 根據 select 框中的 Code 選取資料並渲染
      • 修改: 根據 select 框中的 Code 選取資料，彈出 Dialog 進行編輯後提交
      • 刪除: 根據 select 框中的 Code 選取資料，顯示於 Dialog 中 (唯讀)，供使用者確認後刪除

使用流程

  1. 開啟專案後，進入 http://localhost:8080/CathayBankDemo/
  2. 新增一筆 JPY 資料 (初始化時未插入 H2)
  3. 依上面功能進行操作
